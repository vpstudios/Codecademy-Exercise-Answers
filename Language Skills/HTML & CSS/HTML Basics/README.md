## 4. Make the head
```html
<!DOCTYPE html>
<html>
<head>
    <title>Hustle Hard</title>
</head>
<body>
    GitHub is cool.
</body>
</html>
```

## 5. Paragraphs in the body
```html
<!DOCTYPE html>
<html>
	<head>
		<title>Hustle Hard</title>
	</head>
<body>
    <p>GitHub is Cool</p>
    <p>Promotion is all</p>
</body>
</html>
```

## 6. Paragraphs and headings
```html
<!DOCTYPE html>
<html>
	<head>
		<title>Hustle Hard</title>
	</head>
<body>
    <h1>It can be anything!</h1>	
    <p>GitHub is Cool</p>
    <p>Promotion is all</p>
</body>
</html>
```
